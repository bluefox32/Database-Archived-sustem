All notable changes to this project will be documented in this file.

## [0.1.0-alpha] - 2026-06-05

### Added
- Matrix Authenticated Compiler core implementation
- Multi-layer authentication (Device/OS/Application)
- P2P node framework (Bluetooth/WiFi ready)
- NFT tokenization engine
- Service unlocker with DAO-style pricing
- Proof of Contribution tracking
- Comprehensive documentation

### Features
- Compiler-based metadata distribution (spec.bin)
- AES-256-GCM encryption with Device Serial lock
- Streaming decryption (codec-like behavior)
- Blockchain-compatible ledger system
- Query-based token allocation
- Multi-user account management

### Security
- Device Serial locking (IMEI pattern)
- Matrix-based key derivation
- HMAC verification
- SHA-256 hashing for content integrity

### Documentation
- Architecture guide
- API reference
- Security model documentation
- Design philosophy explanation
- Deployment guidelines
